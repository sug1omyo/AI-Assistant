[1] Step-3.5-Flash  ⭐⭐⭐⭐⭐  (QUAN TRỌNG NHẤT)
    https://github.com/stepfun-ai/Step-3.5-Flash
────────────────────────────────────────────────────────────────────────────────

ĐÂY LÀ GÌ:
  - Model ngôn ngữ lớn (LLM) open-source mạnh nhất của StepFun
  - Kiến trúc: Sparse MoE, 196B tổng params, chỉ kích hoạt 11B/token
  - Context window: 256K tokens
  - Tốc độ: 100–300 tok/s (peak 350 tok/s cho coding)
  - Benchmark: AIME 2025 = 97.3%, SWE-bench = 74.4%, GAIA = 84.5%
  - So sánh: ngang Claude 4 Sonnet / GPT-4.5 về reasoning và agentic
  - License: Apache 2.0 (dùng thương mại được)

NÓ PHỤC VỤ SERVICE NÀO:
  → services/chatbot/  (thay thế hoặc bổ sung Grok/OpenAI/DeepSeek)

VÌ SAO DÙNG:
  - FREE qua OpenRouter: model "stepfun/step-3.5-flash:free"
  - Hoặc API riêng: api.stepfun.ai/v1 với key riêng
  - Tương thích hoàn toàn OpenAI SDK (base_url khác)
  - Không cần thay code nhiều do đã dùng openai.OpenAI(base_url=...)

CÁCH TÍCH HỢP (ĐƠN GIẢN NHẤT - CHỈ CẦN THÊM VÀO .env):

  # services/chatbot/.env - thêm:
  STEPFUN_API_KEY=your-stepfun-api-key
  # hoặc dùng OpenRouter free:
  OPENROUTER_API_KEY=your-openrouter-key

  # services/chatbot/core/config.py - thêm:
  STEPFUN_API_KEY = os.getenv('STEPFUN_API_KEY')
  OPENROUTER_API_KEY = os.getenv('OPENROUTER_API_KEY')

  # services/chatbot/core/chatbot_v2.py - trong ModelRegistry._initialize_models():
  if STEPFUN_API_KEY:
      self._configs['step-flash'] = ModelConfig(
          name='step-flash',
          provider=ModelProvider.OPENAI,  # dùng lại OpenAI provider
          api_key=STEPFUN_API_KEY,
          base_url='https://api.stepfun.ai/v1',
          model_id='step-3.5-flash',
          supports_streaming=True,
          fallback_model='grok'
      )

  # Hoặc dùng OpenRouter (FREE):
  if OPENROUTER_API_KEY:
      self._configs['step-flash-free'] = ModelConfig(
          name='step-flash-free',
          provider=ModelProvider.OPENAI,
          api_key=OPENROUTER_API_KEY,
          base_url='https://openrouter.ai/api/v1',
          model_id='stepfun/step-3.5-flash:free',
          supports_streaming=True,
          fallback_model='grok'
      )

DEPLOY LOCAL (nếu có phần cứng mạnh - cần 8x GPU H100 hoặc Mac Studio M4):

  vllm serve stepfun-ai/Step-3.5-Flash \
    --served-model-name step3p5-flash \
    --tensor-parallel-size 8 \
    --reasoning-parser step3p5 \
    --enable-auto-tool-choice
  
  # Sau đó trong chatbot, dùng localhost thay vì cloud API:

[2] Step-Audio2  ⭐⭐⭐⭐⭐  (RẤT QUAN TRỌNG)
    https://github.com/stepfun-ai/Step-Audio2
────────────────────────────────────────────────────────────────────────────────

ĐÂY LÀ GÌ:
  - End-to-end multimodal LLM cho audio understanding + speech conversation
  - Hỗ trợ: ASR (nhận dạng giọng nói), Speech-to-Speech, dịch giọng nói,
    phân tích cảm xúc, tool calling qua giọng nói, RAG âm thanh
  - Model sizes: mini (nhỏ), mini Base, mini Think (có reasoning)
  - License: Apache 2.0
  - Benchmark: MMAU = 73.2% (mini), ASR tiếng Anh ~3% WER (tốt hơn Whisper large)
  - Hỗ trợ đặc biệt: tiếng Trung, giọng địa phương, phát hiện cảm xúc

NÓ PHỤC VỤ SERVICE NÀO:
  → services/speech2text/  (thay thế hoặc nâng cấp Whisper)

LỢI THẾ SO VỚI WHISPER HIỆN TẠI:
  - Whisper chỉ làm ASR thuần
  - Step-Audio2: ASR + hiểu ngữ nghĩa + cảm xúc + tool calling + TTS
  - Hỗ trợ voice conversation 2 chiều (không chỉ nhận dạng 1 chiều)
  - Tích hợp web search qua giọng nói
  - API streaming qua vLLM

YÊU CẦU PHẦN CỨNG:
  - Model mini: 7B params, cần ~14GB VRAM (RTX 3090/4090)
  - mini Think: tương tự
  - Có thể chạy CPU nếu quantize nhưng chậm

CÁCH CÀI ĐẶT:
  conda create -n stepaudio2 python=3.10
  conda activate stepaudio2
  pip install transformers==4.49.0 torchaudio librosa onnxruntime s3tokenizer diffusers hyperpyyaml
  
  git clone https://github.com/stepfun-ai/Step-Audio2
  cd Step-Audio2
  git lfs install
  git clone https://huggingface.co/stepfun-ai/Step-Audio-2-mini

TÍCH HỢP VÀO services/speech2text/:
  # Tạo file: services/speech2text/src/step_audio2_handler.py
  import sys
  sys.path.insert(0, '/path/to/Step-Audio2')
  from stepaudio2 import StepAudio2
  
  class StepAudio2Handler:
      def __init__(self, model_path='Step-Audio-2-mini'):
          self.model = StepAudio2(model_path)
      
      def transcribe(self, audio_path):
          return self.model.transcribe(audio_path)
      
      def chat_with_audio(self, audio_path, text_context=None):
          return self.model.chat(audio=audio_path, context=text_context)

  # Chạy web demo riêng:
  pip install gradio
  python web_demo.py  # sẽ có UI trên localhost

================================================================================
PHẦN 5: GỢI Ý TỰ CHẾ TẠO MODEL RIÊNG
================================================================================

MỤC TIÊU: Tạo model tiếng Việt riêng, hiểu context chatbot của bạn

BƯỚC 1: Thu thập dữ liệu
  - Lấy conversation history từ MongoDB (đã có ConversationDB)
  - Format: {"messages": [{"role": "user", "content": "..."}, {"role": "assistant", "content": "..."}]}
  - Lọc các conversation chất lượng tốt (rating cao, user satisfied)
  - Mục tiêu: 10,000+ pairs để fine-tune hiệu quả

  # Script thu thập data:

  def export_for_finetuning(min_messages=3, quality_threshold=0.7):
      conversations = ConversationDB.get_all()
      training_data = []
      for conv in conversations:
          messages = MessageDB.get_by_conversation(conv['_id'])
          if len(messages) >= min_messages:
              training_data.append(format_for_training(messages))
      return training_data
